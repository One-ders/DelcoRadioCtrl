# DelcoRadioCtrl GPIO Pin Mappings

This document outlines the GPIO pin mappings specific to the DelcoRadioCtrl project.

## Pin Mappings

| Function            | Pin   |
|---------------------|-------|
| IGNITION            | PA9   |
| TUNE_WHEEL         | PB0   |
| BUTTON_INPUT_1     | PC1   |
| BUTTON_INPUT_2     | PC2   |
| TUNER_CONTROL_1    | PD3   |
| TUNER_CONTROL_2    | PD4   |
| DISPLAY_CONTROL_1   | PE5   |
| DISPLAY_CONTROL_2   | PE6   |

Ensure to verify the pin mappings in the code and hardware setup as per the device specifications.
